# Purchasing Power of Poles Analysis (2014–2024)
##📌 Project Overview
The goal of this project was to analyze the macroeconomic changes in Poland over a decade (2014–2024) and assess their direct impact on citizens' everyday financial reality. Instead of relying solely on dry inflation indicators, I designed a two-page interactive dashboard that translates complex macroeconomic data (Wages, HICP Inflation, and Currency Rates) into a tangible, real-world metric: the "Bread Test" (the real purchasing power of an average wage).
